E-Cubes: A Dissemination, Teaching, and Training Framework

Project Overview

There is a need for easy-to-install, easy-to-use, and easy-to-support educational materials that can be distributed to different individuals and communities lacking access to proper cyber resources and the support of dedicated IT staff. Addressing this need, we introduce E-Cubes, a customizable, extensible, and convenient educational framework, which is deployed as open-access, plug-and-play, and hands-on learning modules. [1]

E-Cubes is available via the Open Source via CC BY (Creative Commons) licensing option as a series of self-contained modules featuring four layers: [1]

  • The Container layer: Provides enhanced convenience for installing E-Cubes in user-specific hardware and software platforms leveraging OS-level virtualization. [1]
  • The Tools layer: Provides the necessary back-end software platforms supporting the educational tasks. [1]
  • The Payload layer: Contains the educational materials for the learning experience, such as course assignments and lab sessions. [1]
  • The Support layer: Contains additional materials like tutorials, videos, and documentation websites to assist students. [1]
